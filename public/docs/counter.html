Final Countdown

A simple and html agnostic date countdown plugin for jQuery

Official Documentation

Please Note: we already added plugin CSS and JS file in vendors.bundle.{css/js}. you can exclude it by using gulp task. you also can add them manually if you want.

How it works

data-countdown="2020/12/12" initialize the countdown also works as wrapper container.

  • .countdown-days // Show days
  • .countdown-hours // Show hours
  • .countdown-minutes // Show minutes
  • .countdown-seconds // Show seconds
